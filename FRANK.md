# FRANK

## Что такое FRANK

FRANK — локальный AI Copilot для продуктового дизайнера.

Главная задача FRANK — автоматизировать рутинную работу продуктового дизайнера.

FRANK не заменяет дизайнера.

FRANK помогает анализировать задачи, принимать UX-решения и генерировать интерфейсы.

## Главные принципы

* Простота важнее идеальности.
* Не реализовывать функциональность заранее.
* Один запрос = одна законченная функция.
* Не менять архитектуру без необходимости.
* Предлагать простые решения.
* Если есть сомнения — сначала спросить пользователя.
